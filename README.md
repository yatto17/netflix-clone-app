# Netflix clone

React・TypeScript を使用して、Netflix の clone アプリを作成いたしました。

## 概要

このアプリは、以下の仕様となっております。

- TMDB から NetflixApi を取得して表示する
- ジャンル別に列に表示
- スライドによってバナーの背景が変わる。
- 映画をクリックすると、映画がある場合は、プロモーション用の YouTube ビデオが iFrame に表示されます。
